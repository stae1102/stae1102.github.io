# 2. 인터넷의 원리

기본적으로 인터넷을 구성하기 위해서는 최소 2 대의 컴퓨터가 필요하며 각각 웹 브라우저 및 웹 서버로 이루어져 웹 브라우저는 프로그램이 설치된 PC로서 웹 서버에 **요청(Request)**를 하며, 웹 서버는 웹 브라우저의 요청에 대해 **응답**한다.  
  
이때, 웹 브라우저를 client, 웹 서버를 server로 지칭하게 된다.
